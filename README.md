# Block-Cracker-Prototype

This is a prototype project of a ***Block Cracker Game*** made in Unity3D. The goal of the project was to simply build a game out of practice.

The project has many features like different blocks that give the player different abilities, player life manager and data persistence for saving the highest score. The movement of the balls are managed using Unity's built in physics engine. The paddle controller was built using the ***Input Manager***, as this is the active input handling thought in the Unity's *Junior Programmer Pathway*.
